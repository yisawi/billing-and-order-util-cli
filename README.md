# Billing & Order Utility CLI

A command-line interface (CLI) tool built in Go to process store billing, calculate taxes, apply conditional shipping discounts, and format final receipts. 

Beyond its functional purpose, this repository serves as a **structured learning environment** to master fundamental and intermediate Go programming concepts, specifically focusing on control structures, memory efficiency, and function design.

---

## 🚀 How to Run

*Instructions will be updated as the CLI interactive features are built.*

```bash
# Clone the repository
git clone [https://github.com/yisawi/billing-and-order-util-cli.git](https://github.com/yisawi/billing-and-order-util-cli.git)

# Navigate to the project directory
cd billing-and-order-util-cli

# Run the CLI tool
go run main.go
